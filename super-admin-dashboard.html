Super Admin SOP

1. Global View

Use Global View for end-to-end portfolio health and workload trend monitoring.

Escalate anomalies when stage counts diverge from expected operational flow.

2. Admin Management

Approve/deactivate privileged accounts based on governance policy and role separation.

Never assign elevated roles without documented authorization trail.

3. Uploader Routing Rules

Define permanent reviewer, RSA, and payment ownership for each uploader where required.

Confirm routing changes do not block users who are not explicitly configured.

4. Security, Audit, Round Robin, Settings

Use audit/security controls for compliance visibility and incident investigation.

Change round-robin and system settings only with approved change-control process.
