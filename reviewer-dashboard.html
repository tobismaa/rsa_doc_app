Reviewer SOP

1. Pending Review

Open Pending Review, inspect customer details and all uploaded documents before deciding.

Always validate file clarity, consistency, and required documents completeness.

2. Approve Action

Use Approve only when submission meets policy and document quality standards.

Approved records move forward to RSA stage; verify customer name and document set before confirming.

3. Reject Action

Use Reject with a clear comment describing exact corrections required.

Keep comments actionable so uploader can fix quickly without repeated rejection cycles.

4. Viewer and Zoom Controls

Use document viewer tools (zoom, fit, next/prev file) to inspect signatures, dates, and numbers.

Reject blurry or incomplete scans that create audit/compliance risk.
