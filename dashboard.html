Uploader SOP

1. New Submission

Start with New Submission, complete customer details, and click Save Details before uploading any file.

Upload required documents one by one or with Batch Upload, then submit all documents when counters show readiness.

2. Track Application

Use Track Application from the header to search by application ID or customer name.

Confirm stage, status, comments, and who has worked on the file before escalating to another team.

3. Loan Calculator

Use Loan Calculator to quickly estimate 25% RSA contribution, facility fee, property value, and loan amount.

This is a guide tool; always submit customer data values from validated source documents.

4. Register Agent

Open Register Agent, click Register New Agent, and fill full name, phone, account number, and bank.

Agent stays pending until admin approval; only approved agents appear in submission dropdown.

5. Paid Customers

Use Active Commission and Cleared Commission subtabs to monitor payouts and settled commissions.

Filter by dates and customer/PFA to reconcile records and confirm payment lifecycle.

6. Rejected Items

Open rejected records, read reviewer comments, correct files, and re-upload using the correction action.

After correction submission, recheck status from Pending and Track Application.
